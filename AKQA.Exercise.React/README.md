Install the dependencies:
0. Install npm if you do not have it
1. Navigate the folder by terminal in Mac/Linux, or NodeJS Bash in Windows
2. Run "npm install"

To run this project:
1. Navigate the folder by terminal in Mac/Linux, or NodeJS Bash in Windows
2. Run "npm run dev"
3. Project will run at http://localhost:8080/

To run test:
1. Navigate the folder by terminal in Mac/Linux, or NodeJS Bash in Windows
2. Run "npm test"
